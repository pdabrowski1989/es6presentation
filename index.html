Block-Scoped Declarations

ES5

var a = 2;
(function IIFE(){
    var a = 3;
    console.log( a ); // 3
})();
console.log( a ); // 2
ES6

var a = 2;
{
    let a = 3;
    console.log( a ); // 3
}
console.log( a ); // 2

Let and Const

  • let statement declares a block scope local variable
  • const, declaration creates a read-only reference to a value


function test() {
  const a = 2;

  // a = 6; -> ERROR

  console.log(a);
}
